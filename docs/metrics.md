## Технические детали

### Архитектура
- **ML-модель:** MobileNetV3-Small (ONNX, FP32)
- **Рендерер:** WebGL с шейдерами (fallback на WASM)
- **Предобработка:** Canvas 2D API (resize до 224×224, нормализация ImageNet)
- **Постобработка:** WebGL fragment shader (яркость, контраст, насыщенность)

### Оптимизации
- Единый проход по пикселям в препроцессоре
- Кэширование uniform locations в WebGL
- Переиспользование буферов геометрии
- Lazy loading модели (по первому запросу)
- Удаление лишних WASM файлов из бандла

### Размер компонентов
- ONNX модель (FP32): 4.11 МБ
- ONNX Runtime WASM: ~1.5 МБ
- JavaScript бандл: ~150 КБ
- CSS стили: ~15 КБ
- **Итого: 5.81 МБ**
